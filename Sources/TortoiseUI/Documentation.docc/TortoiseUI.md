# ``TortoiseUI``

SwiftUI rendering and animation for TortoiseGraphics.

## Overview

`TortoiseUI` consumes a ``TurtleCommand`` stream from ``TortoiseCore``
and renders it as an animated SwiftUI view using `TimelineView` and `Canvas`.

Features:
- Animated playback with configurable speed
- Step-by-step execution
- iOS / iPadOS / macOS / visionOS support
